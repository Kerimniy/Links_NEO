Приложения с GitHub

KHA-331_APP - приложение для хэширования вашей строки. KHA-331 - хэш алгоритм генерирующий хэш длиной 331 бит (64 символа, 0-9, a-z) на основе побитовых операций, реализован на C# и Python.

Bindm. Простое приложение, которое может регулировать громкость системы (повышать, понижать громкость или вообще отключить звук) с помощью комбинации функциональных клавиш (от 1 до 3) и одной любой клавиши (например, CTRL + SHIFT + ALT + H).

Bindm. Простое приложение, которое может запускать любое (почти) приложение, открыв его в проводнике, с помощью комбинации функциональных клавиш (от 1 до 3) и одной любой клавиши (например, CTRL + SHIFT + ALT + H).

Игры на Unity

The Space Seeker - игра, где ты на космическом корабле ищешь потерянные запчасти среди астероидов. На данный момент игра доступна для скачивания на Itch.io или Яндекст играх [Снято с публикации]. Игра поддерживает только десктопные платформы.

Симулятор управления АЭС [В разработке] - игра где ты управляешь атомной электростанцие, изменять мощность, чтобы она соответсвовала текущей квоте. Управление осуществляется регулирующими стержнями и циркуляционными насосами. На данный момент игра доступна на Яндекст играх [Черновик]. Игра поддерживает как десктоп, так и мобильные платформы.
